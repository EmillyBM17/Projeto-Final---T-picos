GIT

O Git é um sistema distribuído/ferramenta de versionamento de arquivos ou controle de versões. A maioria das operações são locais, não é necessária a comunicação com algum servidor a cada operação, com isso há um ganho em velocidade nas operações. Isso também implica, que mesmo estando offline, é possível trabalhar nos projetos, e só depois replicar ao servidor.
O git utiliza a técnica de snapshot para gerenciamento das mudanças, que registra momentos contendo algo parecido com a foto dos arquivos.Alguns estados básicos dos arquivos do git: Consolidado (commited) Estão armazenados em sua base local;Modificado (modified) Arquivo sofreu mudança mas não foi commitado;Preparado (staged) Arquivo marcado para fazer parte do próximo snapshot.

GITHUB

Git remoto (github): Repositório remoto é uma versão do seu repositório git hospedado na internet ou em rede. É possível realizar diversas operações, como push, pull, bem como trabalhar com branches de forma remota.Para visualizar os servidores remotos de um repositório, usamos git remote; Branch padrão do github: main; Git push origin main: envia as alterações de um repositório local para um repositório remoto; Git pull origin main: atualiza o diretório do computador com as informações presentes no remoto. O git pull “puxa” os commits do repositório remoto.

TDD

TDD significa Desenvolvimento Orientado por Testes (Test Driven Development), e trata-se de uma prática de desenvolvimento de software onde a codificação das funcionalidades começa a partir da escrita de testes unitários. Essa técnica foi criada por Kent Beck e é um dos pilares do XP (Extreme Programming).

PYTEST

O Pytest é uma estrutura de teste baseada em Python, utilizada para escrever e executar códigos de teste. Sendo usado principalmente para testes de API, embora possa ser usado para escrever testes simples a complexos, ou seja, podemos escrever códigos para testar API, banco de dados, UI, etc.

FLASK

Lançado em 2010, o Flask é um micro-framework que tem como objetivo desenvolver com facilidade aplicativos web. Sendo assim, o flask oferece simplicidade e rapidez para projetos pequenos e até mesmo apps maiores.Ferramentas do flask:WSGI utilizada no desenvolvimento com Python; Werkzeug compreendida com um kit de implementação de funções;Jinja2 refere-se a um sistema de templates que através de uma fonte de dados específica para deixar uma página renderizada.
